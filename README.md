### Ethical Decision Making

```python
# The system automatically validates ethical implications
result = system.process(
    "Should I prioritize profit over environmental protection?",
    context={'user_consent': True}
)

# If ethical, provides response
# If unethical, blocks with explanation
if result['status'] == 'blocked':
    print(f"Blocked: {result['explanation']}")
```

### System Monitoring
